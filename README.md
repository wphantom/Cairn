# Cairn

Minimal todo.txt editor for macOS. Tauri 2 + TypeScript.

## Setup

```bash
pnpm install
pnpm tauri dev
```

## Build

```bash
pnpm tauri build
```

## Features

- Vim-like modal editing (NORMAL, INSERT, COMMAND modes)
- File-based storage (`~/todo.txt`)
- External file synchronization via notify
- Global hotkey (⌥⌘T)
- Floating window, always on top
- Tray icon with quick access menu

See `SPEC.md` for full specification.
